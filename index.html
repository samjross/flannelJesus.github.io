Update Author Attributes

In _config.yml remember to specify your own data:

{% highlight yaml %} title : My Blog =) author : name : Name Lastname email : blah@email.test github : username twitter : username {% endhighlight %}

The theme should reference these variables whenever needed.

Sample Posts

This blog contains sample posts which help stage pages and blog data. When you don’t need the samples anymore just delete the _posts/core-samples folder.

{% highlight console %} $ rm -rf _posts/core-samples {% endhighlight %}
